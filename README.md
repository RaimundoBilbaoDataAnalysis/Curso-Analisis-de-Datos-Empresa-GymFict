# GymFict - Curso de Análisis de Datos (Curso Completo)

¡Bienvenido al repositorio oficial del canal! Aquí encontrarás todos los recursos, bases de datos y soluciones que utilizaremos a lo largo de nuestra ruta de aprendizaje desde cero hasta nivel profesional.

## 🏢 El Caso de Negocio: GymFict.
### Debes tener en cuenta que es una empresa ficticia, todo lo relacionado a la empresa es ficticio, fue creado con el fin de poder realizar este curso.
**GymFict** es una cadena emergente de tiendas de artículos deportivos, indumentaria fitness y suplementación alimenticia con un canal de ventas híbrido (E-commerce y tiendas físicas). 

A medida que avances en las listas de reproducción del canal, la empresa crecerá y tus responsabilidades como su Analista de Datos principal evolucionarán:
1. **Fase 1 (Excel):** Analizarás las ventas del primer año de operaciones usando una tabla plana para generar reportes rápidos a la gerencia.
2. **Fase 2 (SQL y Power BI):** Migrarás los datos a un modelo relacional para limpiar millones de filas e inconsistencias.
3. **Fase 3 (Python & Advanced Data):** Automatizarás reportes consumiendo datos desestructurados y APIs logísticas.
